Mô tả:
    I. Cách thức cài đặt/biên dịch/chạy chương trình. Đầu tiên giải nén project nằm trong file flow-free-solver-master.zip
        1. Sử dụng Eclipse import project và bấm run Run
        2. Mở trực tiếp bằng file jar: double click vào file jar để chạy chương trình (Có khả năng không hoạt động)
        3. Mở bằng terminal/command prompt: cd đến root của project (ls/dir trên terminal/command prompt thấy có chứa FlowFree.jar). Chạy: java -jar FlowFree.jar
    II. Gói phần mêm, thư viện kèm theo: AbsoluteLayout và Java Swing.
